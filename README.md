# NexEd – A Modern School Management App 📚✨

**NexEd** is a sleek, user-friendly school management app designed to simplify and streamline school administration for teachers, students, and parents. This powerful tool enables seamless communication, efficient management of academic records, and real-time access to school events – all wrapped in a beautiful, intuitive interface.

### Features 🛠️
- **Student Information Management**: Easily manage student profiles, academic records, and attendance tracking.
- **Teacher and admin Portal**: Streamlined access for teachers to manage classes, assignments, and track student performance.
- **Parent Dashboard**: Real-time access to student grades, attendance, and important announcements.
- **Assignment and Exam Scheduling**: Stay organized with a dedicated module for assignment submissions and exam scheduling with notifications.
- **School Event Calendar**: View and manage school events, holidays, and announcements in a unified calendar.
- **Communication Center**: In-app messaging to bridge communication gaps between teachers, students, and parents.
- **Mobile-Optimized & Responsive**: Access the app from any device with a beautiful, mobile-first design.

### Tech Stack 💻
- **Frontend**: Next.js, React, Tailwind CSS for a responsive and modern UI.
- **Backend**: Next.js,Prisma as the ORM for database management.
- **Validation**: Zod for schema validation and data validation.
- **Icons**: Lucide for clean and customizable iconography.
- **Calendar**: React Calendar for intuitive scheduling and event management.
- **Charts**: Smart Chart Library for visual data insights and analytics.
