  <h1>NexEd – A Modern School Management App 📚✨</h1>
  <p><strong>NexEd</strong> is a sleek, user-friendly school management app designed to simplify and streamline school administration for teachers, students, and parents. This powerful tool enables seamless communication, efficient management of academic records, and real-time access to school events – all wrapped in a beautiful, intuitive interface.</p>

  <h3>Features 🛠️</h3>
  <ul>
    <li><strong>Student Information Management</strong>: Easily manage student profiles, academic records, and attendance tracking.</li>
    <li><strong>Teacher and admin Portal</strong>: Streamlined access for teachers to manage classes, assignments, and track student performance.</li>
    <li><strong>Parent Dashboard</strong>: Real-time access to student grades, attendance, and important announcements.</li>
    <li><strong>Assignment and Exam Scheduling</strong>: Stay organized with a dedicated module for assignment submissions and exam scheduling with notifications.</li>
    <li><strong>School Event Calendar</strong>: View and manage school events, holidays, and announcements in a unified calendar.</li>
    <li><strong>Communication Center</strong>: In-app messaging to bridge communication gaps between teachers, students, and parents.</li>
    <li><strong>Mobile-Optimized & Responsive</strong>: Access the app from any device with a beautiful, mobile-first design.</li>
  </ul>
